## Requirements
``1. python3``


``2. mpv [for streaming]``


``3. ffmpeg [for downloading]``



***
## Usage

1. ``pip install -r requirements.txt``
2. ``python movie_app.py`` [to stream and download movies]
3. ``ptyhon series_app.py`` [to stream and download tv series]
